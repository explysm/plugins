# Plugins

Welcome to the plugin list. This repo contains several plugins for Vendetta and Revenge mods.

## [AutoNote](https://github.com/explysm/plugins/tree/master/plugins/autonote)
**Description:** Automatically adds custom notes or headers to messages when triggered by specific keywords at the beginning of a message.
- **Key Features:**
  - Multiple auto-note profiles.
  - Trigger word anchoring (only triggers at the start).
  - Custom footers with dynamic placeholders (`{trigger}`, `{time}`, `{date}`).
  - Auto-styling (subtext, blockquote, code block).
  - Flexible positioning (top or bottom).
  - **Custom JavaScript Scripting** for advanced users.
- **Install Link:** `https://explysm.github.io/plugins/autonote`

## [RealmojiExtended](https://github.com/explysm/plugins/tree/master/plugins/realmojiextended)
**Description:** Makes freemojis real, including animations and jumbo emoji support.
- **Key Features:**
  - Converts emoji URLs into real emoji embeds.
  - Supports animated emojis (GIF/WebM).
  - Jumbo emoji support for standalone emojis.
- **Install Link:** `https://explysm.github.io/plugins/realmojiextended`
