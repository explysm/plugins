# Vendetta Plugins Template
This repo contains plugins for [Revenge](https://github.com/revenge-mod/revenge-bundle) and other various vendetta based mods.

>[!IMPORTANT]
>These plugins are edited by ai using LLMs such as:<br>
>Gemini and Big Pickle<br>
>Plugins are tested to ensure reliability.<br>
>These are forks of original plugins to add new features & fix bugs that otherwise wouldn't be fixed.


# How to install?
Paste a plugin URL into the Plugins page of FireCord, following a basic format of:

https://`explysm`.github.io/`fire-plugins`/`PLUGIN_NAME`
